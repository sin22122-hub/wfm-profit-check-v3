# PFM V22B RC1｜Professional Report

## 版本定位
V22B RC1 是 V22 的 Professional Report 版，目標是讓目前結果頁與 PDF 報告達到可正式 Demo、可對外測試的顧問報告水準。

本版沒有修改 Google Sheet 公式引擎；分數邏輯仍待 V23 Decision Engine 重構。

## 新增 / 調整

### 1. 三大問題 / 三大優勢
- 三大問題加入分類標題，例如：財務風險、服務定價、固定成本、客戶經營等。
- 三大優勢加入分類標題，例如：客戶信任、獲利優勢、回流基礎等。
- 星等取消背景圈，改為單純紅 / 綠星星呈現。
- 風險、第一步、放大方向、PFM 顧問提醒改為顧問式區塊。
- 問題使用淡紅色系，優勢使用淡綠色系。

### 2. 雷達圖 V2
- 保留雷達圖。
- 新增五大能力卡：獲利能力、客戶經營、流量能力、轉換能力、品牌成熟。
- 每張能力卡包含：分數、評級、評估重點、PFM 顧問建議。
- 不公開公式權重與計算細節，保留 PFM know-how。

### 3. PFM 商業診斷可信度
- 移到完整報告最後，不再放在 Hero / 前段。
- 不再使用 A+、A、B+ 等容易與店家分數混淆的評級。
- 改為「分析信心高 / 良好 / 中等」等語意。
- 補充說明：此指標代表資料完整度與分析信心，不等於店家經營分數。

### 4. PDF 同步
- PDF 第 07 頁三大問題同步新版結構。
- PDF 第 08 頁三大優勢同步新版結構。
- PDF 第 09 頁成長潛力藍圖加入新版能力卡。
- PDF 最後頁加入 PFM 商業診斷可信度。
